Quick Edit Guide

How to add a Scratch embed

  1. Open projects.js.
  2. Duplicate one project object in the scratchProjects array.
  3. Paste Astra’s shared Scratch embed code into the embedHtml field.
  4. Update the title, summary, and tag fields.
  5. Save and republish the site.
{
  title: "New Project Title",
  summary: "Short description of what Astra made.",
  tag: "Animation",
  embedHtml:
    '<iframe src="https://scratch.mit.edu/projects/123456789/embed" allowtransparency="true" width="485" height="402" frameborder="0" scrolling="no" allowfullscreen></iframe>'
}
